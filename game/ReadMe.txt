◆動作環境目安
　OS : Windows Vista　7　8　8.1
　DirectX : DirectX9.0c以上
　CPU : Intel Core2Duo 2.6GHz相当以上
　メモリ : 2GB以上の搭載メモリ
　HDD：1GB以上の空き容量
　グラフィックカード : DirectX9.0c、シェーダー2.0に対応したVRAM512MB以上のもの (※オンボードグラフィックは動作対象外です)
　ディスプレイ : 960×720の解像度を表示可能なもの
　入力デバイス：ゲームパッド（推奨）・キーボード


◆インストール・アンインストール・起動・更新
　インストール　　：setup.exeを起動してください。
　アンインストール：コントロールパネルの「プログラムの変更と削除」から幻想人形演舞を選択してください。
　セーブデータやスクリーンショットは残りますので、不必要な際はアンインストール後、手動で削除してください。


◆起動の前に
　更新がある場合がありますので、「http://www.fo-lens.net/gn_enbu/」から最新パッチが出ていないか必ずご確認ください。


◆セーブデータについて
　セーブデータはインストールフォルダの中の「save」フォルダの中にある、
　「gn_enbu.sav」です。セーブデータを消去したい場合は、これを削除してください。
　なお、体験版のセーブデータは、完成版では使用できません。


◆ストーリー
　暇をした主人公は、神隠しがあると噂されている博麗神社へと向かう。
　途中のバスの中、『八雲紫』と名乗る少女と出会う夢を見たのだが・・・


◆操作方法（括弧内はキーボードキー）
　方向キー（↑↓←→キー）		メニューカーソル/プレイヤーの移動
　Aボタン（Zキー）				決定/目の前を調べる
　Bボタン（Xキー）				キャンセル/押しながらプレイヤー移動で走る
　Xボタン（Cキー）				ポーズ表示等
　Yボタン（Sキー）				道具ソート・人形箱で複数選択
　Lボタン（Aキー）				メニューページ切り替え等
　Rボタン（Dキー）				メニューページ切り替え等
　STARTボタン（スペースキー）	ポーズ表示等
　SELECTボタン（左シフトキー）	道具ソート等

　対応ボタンなし（F11キー）		画面のウィンドウ/フルスクリーンを切り替えます。
　対応ボタンなし（Pキー）		スクリーンショットを保存します。
　								保存先は「save」フォルダ内の「screenshot」フォルダです。
　
　ウインドウモード時、画面を左ダブルクリックすることで、一定の倍率で画面が大きくなります。
　また、画面を右ダブルクリックでは、一定の倍率で画面が小さくなります。
　ウィンドウの縁をドラッグすることで画面サイズを変更することもできます。


○ヒント
　フィールドにいるキャラクターや看板から戦闘についてのヒントを得られる場合があります。
　キャラクターには積極的に話しかけ、看板なども見逃さないようにしましょう。
